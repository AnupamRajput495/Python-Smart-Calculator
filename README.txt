# Python Calculator

A basic calculator program in Python that handles mathematical expressions following the order of operations (BODMAS). The program allows the user to input expressions, performs the calculations, and displays the result.

## Features

- Handles addition, subtraction, multiplication, and division operations.
- Supports expressions with or without spaces between operands and operators.
- Checks for division by zero and displays an error message when encountered.
